## 📚프로젝트 소개
제작기간 : 2024.03.04 ~ 2024.03.14 <br>
제작인원 : 3명

- 프로젝트를 추가하여 직원들의 프로젝트 공유가 가능한 ```캘린더```기능 제공 
- 근태를 신청하여 휴가관리를 할 수있는  ```게시판``` 제공
- 관리자는 직원들 ```근태신청```을 관리하고 ```급여```를 지급할 수 있는 ```관리자 권한``` 제공
- ```실시간채팅```이 가능한 ```채팅``` 기능제공
- ```반응형```(PC, 태블릿, 휴대폰) 제공
